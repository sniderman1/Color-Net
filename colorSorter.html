Color Net

Welcome to Color Net. This is a program that uses a JavaScript neural net called Brain.js. Brain.js is an open source neural net that was developed by MIT and is available to everyone on NPM.

Color Net uses the user input to train the neural net in color detection. You, the user, will be shown a serious of colors and be asked to rate them from 0% to 100%. You can pick whatever color you want. For example, say you pick red. You decide how close each random color that will appear on screen is to red. Once you have completed 15 rounds there will be a button that says, "train net." Once you click this button a serious of functions will take place. First, the neural net will be fed all the data you just inputted. This trains the neural net. Next, the net will receive a bunch of random colors, much like you did in the test. As the neural net is receiving these colors it is giving an output on each of the colors. The output is between 0 and 1, which translates from 0% to 100%. If the output is over 65% the color is kept. This will take place 60 times. Once 60 colors have been picked, all colors will be displayed at the bottom of the page with the percent value the neural net gave to them.
